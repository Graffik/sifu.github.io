CHP2630/2635

  • General Description
  • CHP2630 is a single-channel 10MHz four phase 12A DCDC buck converter.
  • CHP2635 is a dual-channel dual phase 5A DCDC buck converter.
  • This IC is designed as a stand-alone component without integrated passives, resulting in the smallest possible footprint.
  • The Dynamic Voltage Control (DVC) supports adaptive adjustment of the output voltage depending on the load. The DVC can be controlled either via direct register writes through the communication interface (I2C or SPI) or via input pins.
  • A remote sensing feature is included to provide high accuracy and to support various PCB routing schemes.
  • Integrated protection features for increased system reliability.
152nm, 0.35u WLCSP ES Silicon Validated
Unisoc 5G Platform
  • Key Features
  • Programmable Switching Frequency Up to 10MHz
  • Input Voltage Range: 2.7V - 5.5V
  • Programmable Output Voltage Range: 0.5V-3.3V
  • Flat and high efficiency over wide load range, up to 92%
  • Output Voltage Step Size 3.125mV
  • Output Current capability:
    • CHP2630: 12A (Single buck)
    • CHP2635: 6A*2 (Dual bucks)
  • Fast DVC with Slew Rate: 250mV/us
  • Control interface: I2C & SPI
  • Automatic, hysteresis-less PFM/PWM transitions
  • Remote sensing at point-of-load
  • Power-Good output status report
  • Programmable OCP, UVLO, Over-Temperature warning and protection
  • Integrated OTP registers
  • WLCSP-64,2.9x2.9mm, 0.35mm ball pitch
